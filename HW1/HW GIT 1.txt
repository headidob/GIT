1. Перед копированием репозитория с GitHub нужно с копируемого репозитория скопировать SSH ключ  
Команда для копирования репозитория с GitHub "git clone скопированный ключ с GitHub" команда производится 
в папке в которую планируется скопировать репозиторий

2. Находясь в папке которую планируется создать как репозиторий на GitHub, нужно ввести команду "git init". 
Создать новый пустой (без ридми файла) репозиторий на GitHub, и скопировать линк SSH и сделать команду 
"git remote add origin скопированный линк"
потом
"git branch -M main"
и потом
"git push -u origin main"
или все команды можно сразу скопировать из подсказки находятсь в пустом репозитории на GitHub

3. Обновления на локальный копьтер подтягиваются c GitHub через команду "git pull" нужно находиться 
в папке залинкованной с GitHub

4. "git status"
показывает статус файлов из папки какие еще не переведены в индекс, 
а какие переведены или если уже есть подготовленный коммит

"git log"
показывает произведенные коммиты

"git remote -v" 
проверяет действительно ли что репозиторий GIT и показывает что можно делать fetch & push на сервер
